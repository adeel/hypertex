The stable motivic homotopy category

A. Khan Yusufzai

1.  
Fix a noetherian scheme $S$ and write $\Sch(S)$ for the essentially small category of finite type $S$-schemes. Let $\Sm(S)$ denote the full subcategory of smooth and finite type $S$-schemes.
2.   Definition. —
Consider the cocomplete category $\SPSh(\Sm(S)) = \PSh(\Sm(S)), \SSet)$ of simplicial presheaves on $\Sm(S)$ (). Recall that the projective model structure on this category has objectwise weak equivalences and objectwise fibrations, where we consider $\SSet$ with the standard model structure (); see (). Note that the fibrant objects are simplicial presheaves that take values in Kan complexes. This model category presents the $(\infty,1)$-category of $(\infty,1)$-presheaves on $\infSm(S)$.
3.   Definition. —
Consider the Nisnevich topology on the category $\Sm(S)$ (???). Recall that one has a Nisnevich-local projective model structure on $\SPSh(\Sm(S))$, where fibrant objects are presheaves that take values in Kan complexes and satisfy descent with respect to Nisnevich hypercovers (). This model category presents the hypercompletion of the $(\infty,1)$-category of $(\infty,1)$-sheaves on $\infSm(S)$. The homotopy category with respect to this model structure, which we will denote by $\Spc(S)$, is called the category of motivic spaces over $S$. Note that there is a canonical functor \[ \L_S : \Sm(S) \to \Spc(S) \] which maps a scheme $X$ in $\Sm(S)$ to the motivic space $\L_S(X)$ which maps $Y \in \Sm(S)$ to the constant simplicial set $\Hom_{\Sm(S)}(Y, X)$. Indeed note that this presheaf is clearly fibrant with respect to the Nisnevich-local projective model structure.
4.   Definition. —
The $\A^1$-local projective model structure on $\SPSh(\Sm(S))$ is the left Bousfield localization of the Nisnevich-local projective model structure at the class of morphisms \[ \L_S(X \times_S \A^1_S) \too \L_S(X) \] induced by the projections $X \times_S \A^1_S \to X$, for $X$ in $\Sm(S)$. The homotopy category with respect to this model structure, which we will denote by $\H(S)$, is called the unstable motivic homotopy category over $S$.
5.   Definition. —
Let $T$ be a cofibrant replacement of the Tate sphere, the motivic space associated to the homotopy cofibre of the morphism \[ \L_S(\Gmult_S) \too \L_S(\A^1_S) \] induced by the inclusion $\Gmult_S \hook \A^1_S$, where $\Gmult_S$ denotes the complement of the point $0$ in the affine line $\A^1_S$.
6.   Definition. —
Consider the category $\Spt_T^\Sigma(\SPSh(\Sm(S)))$ of symmetric $T$-spectra in $\SPSh(\Sm(S))$ (). Recall that it has a stable projective model structure induced by the $\A^1$-local projective model structure on $\SPSh(\Sm(S))$ (Ayoub, 4.3.29). The homotopy category associated to this model structure is called the stable motivic homotopy category over $S$, denoted $\SH(S)$. We write \[ \M_S : \Sm(S) \too \SH(S) \] for the composite of the functor $\L_S : \Sm(S) \to \H(S)$ with the localization functor $\H(S) \to \SH(S)$.
